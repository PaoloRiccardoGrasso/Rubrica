Gestione Rubrica CLI 📖

Status: In Progress

✨ Features

  • 📜 View Contacts: Organized and clear contact display.
  • ➕ Add Contacts: Add new contacts with details like name, phone, and email.
  • ✏️ Edit Contacts: Update existing contact details easily.
  • 🗑️ Delete Contacts: Securely remove unwanted contacts.
  • 🔍 Search Contacts: Quickly find contacts using keywords or criteria.
